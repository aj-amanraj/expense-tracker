# Expense Tracker

A web application built using React, Node.js, Express, and MongoDB.
This project helps users track their daily expenses, categorize spending, and manage personal finances.

## Features

- User Signup and Login Authentication
- JWT based Authentication
- Protected Routes
- Add Expenses
- View Expense List
- Category Based Expense Summary
- Responsive Dashboard UI
- Secure Password Hashing using bcrypt

## Tech Stack

### Frontend
- React (Vite)
- React Router DOM
- Tailwind CSS
- Material UI

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcryptjs

## Installation

### Clone Repositry
  - git clone https://github.com/aj-amanraj/expense-tracker.git

### Frontend Setup
  - cd frontend
  - npm install
  - npm run dev

## Backend Setup
  - cd backend
  - npm install
  - npm run dev

## Environment variables (.env file)
  - MONGO_URI=your_mongodb_connection_string
  - PORT=5000
  - JWT_SECRET=your_secret_key
